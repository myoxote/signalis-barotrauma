This is the GitHub repository for the SIGNALIS Replika Jobs mod for Barotrauma, uploaded to Steam Workshop.
Please note that this is a deprecated project, and that this repository exists that others might continue work on the mod.
The code is highkey very messy, and there may be incomplete parts in the repo. I would suggest using the STAR as a base, because it is currently the most "complete" Replika out of the ones available.

I won't be able to answer many questions about the mod because it's been a long while since I've last worked on it, but I sincerely appreciate the support and hope that there can be a complete version of the mod, for everyone's Replikamaxxing needs!!
